# MARL SoccerTwos

### Research project  multi-agent reinforcement learning concepts in a 2-versus-2 soccer environment. Utilized the SoccerTwos env from Unity's ML-Agents Toolkit  

Areas explored:
- Reinforcement Learning
- Multi-Agent Environments
- Cooperation vs. Competition
- Deep Q Learning
- Joint Action Learning
- Evolutionary Strategy

Work completed as a final project submission for Columbia University's EECS6892: Reinforcement Learning
